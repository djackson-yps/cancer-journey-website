# My Cancer Journey — Marketing Website

Source for https://mycancerjourney.co.uk (Netlify project: `mycancerjourney`).

Three static pages: `index.html`, `privacy-policy.html`, `security.html`.

## Deploying
This site is NOT auto-deployed from GitHub. After changing files, either:
1. Ask Claude to deploy (it uses the Netlify MCP), or
2. Drag the three HTML files onto the Netlify deploy page (app.netlify.com → mycancerjourney → Deploys), or
3. Connect this repo to Netlify (Site settings → Build & deploy → Link repository) for automatic deploys.
